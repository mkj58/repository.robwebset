= robwebset Addon Repository =

The "robwebset Reposotory" can be enabled on your Kodi installation by installing the following Addon Zip:

:https://github.com/robwebset/repository.robwebset/raw/master/repos/repository.robwebset/repository.robwebset-1.0.0.zip

Instructions on how to install a repository can be found on the following page:

:[[HOW-TO:Install_add-ons_from_zip_files]]

The robwebset repository stores lots of popular addons created and maintained by robwebset.

= Why? =

Some people may wonder why these addons are not in the official repository. The majority of them used to be, however after several repository rule changes after they were in the official repository I couldn't keep the feature rich functionality in tacked while keeping them in the official repository (Mainly due to a change in Keymap rules). So I decided it would be easier for everyone to just have them in my own repository.
